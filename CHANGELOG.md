# Changelog

## [1.0.1] - 2026-06-20

### What's New
- add language selector to settings
- add localization support with Russian and English
- persist favicon URL and add auto-icon field to config
- replace icon picker with popup menu, add favicon timeout and fallback
- handle empty categories and sites with inline action buttons
- replace FAB with inline buttons in settings, default theme to system

**Full Changelog**: https://github.com/cheack/tabik/compare/1.0.0...1.0.1
